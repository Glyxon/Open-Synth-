# Open-Synth-
Open-Synth is an open-source platform for decentralized Enzymatic DNA Synthesis (EDS). By combining engineered TdT-Car9 enzymes with Digital Microfluidics (EWOD), we enable on-demand production of RNAi triggers and oligos. Developed by Glyxon Biolabs in collaboration with USFQ and IFC-UNAM to democratize genetic infrastructure. #DeSci #Biopunk
